## Featured Work

### [NASA Air Quality Digital Twin](https://github.com/benadark1/DigiTwin)
Earth System Digital Twin predicting AQI from historical sensor data. 8 Random Forest models (R² = 0.94), deployed serverless on AWS Lambda + API Gateway, with an interactive JavaScript dashboard. Built during my NASA/KBR internship.

### AeroTwin: Vision-Based Structural Load Prediction
Three-stage deep learning pipeline (ResNet-18 → Physics-Informed Neural Network → MPC) replacing physical strain gauges with learned surrogate models. R² > 0.99 on 149 NASA CFD simulations from only 105 training samples. *UW ENGR 521, 2026.*

### [Coastal Flooding Prediction](https://github.com/benadark1/Coastal_Flooding_Hackathon)
Time-series ML for sea-level forecasting across multiple sites. NSF HDR x UW Machine Learning Hackathon, January 2026.
